# Poshan Sahayak: AI-Driven Rural Healthcare Assistant

**Poshan Sahayak** is a Python-based Telegram bot designed to bridge the data gap in India's rural healthcare system. Developed as a capstone for my **CS50x** journey, this tool assists **Anganwadi workers** and **Blood Bank managers** in tracking child nutrition and resource allocation.

## 🏆 Achievements & Context
* **National Level Winner**: CSIR Aero Fest
* **National Finalist**: Youth Ideathon (Top 125)
* **Inspired by CS50x**: Built using Python logic and data abstraction principles.

## 🚀 Features
* **Growth Tracking**: Automated malnutrition detection based on age-to-weight ratios.
* **Vaccination Alerts**: WhatsApp notifications for parents regarding upcoming immunization doses.
* **Maternal Health Monitoring**: Tracking Hb levels and BP for pregnant women in rural centers.
* **Inventory Management**: Real-time tracking of ration distribution (Rice/Dal).

## 🛠️ Tech Stack
* **Language**: Python
* **APIs**: Telegram Bot API, Twilio (WhatsApp)
* **Libraries**: Matplotlib (Growth Charts), gTTS (Voice Feedback in Hindi), PIL (Digital ID Generation)

## 🔐 Security Note
*Hardcoded API credentials have been removed and rotated for security. This project is configured to use environment variables for local deployment.*# Poshan-Sahayak
